# afrijeng-server

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```
# database setup
```env
DB_HOST=your_localhost
DB_USER=your_user
DB_PASS=your_pass
DB_NAME=your_db_name
```

This project was created using `bun init` in bun v1.2.12. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
